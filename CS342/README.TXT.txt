The game supports the moves GO direction, GRAB artifact, DROP artifact, USE artifact, EXIT, QUIT, LOOK, INVENTORY,
and NOTHING.

each move type is recognized by three strings: FULL CHAPS, all under chaps, First letter chap. ex:GO,Go,go

all direction strings are supported by project 2 requirments.

UI not supported all UI characters will be asked to make a move.
If user move is invalid user looses turn. 

SUpport for both 4.0 and 3.1 file version
